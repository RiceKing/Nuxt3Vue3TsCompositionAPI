# Песочница на Nuxt 3

Цель повысить навыки: 
Vue3, Nuxt3, TypeScript, Composition API

### Setup

```bash
# npm, node v18.0.0 or newer
npm install

# run dev
npm run dev

# run production
npm run build

# preview production
npm run preview
```
